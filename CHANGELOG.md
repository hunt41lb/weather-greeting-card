# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-01-XX

### Added

- Initial release
- Personalized greeting with `{{user}}` template support
- Weather icon display from entity attributes
- Two customizable stat fields with prefix/suffix support
- Visual editor for all configuration options
- HACS compatibility
- Full grid layout support for sections view
